algorithms.forward_step¶

Module: algorithms.forward_step¶

Inheritance diagram for selection.algorithms.forward_step:

In this module, we implement forward stepwise model selection for \(K\) steps. The main goal of this is to produce a set of linear inequality constraints satisfied by \(y\) after \(K\) steps.
