Compute the number of rejections using forward stop of `http://arxiv.org/abs/1309.5352`_

>>> forward_stop(np.array([0.5,0.6,0.7,0.8,0.9]), 0.05)

>>> forward_stop(np.array([0.001, 0.002, 0.0015, 0.0013, 0.05, 0.6]), 0.05)
